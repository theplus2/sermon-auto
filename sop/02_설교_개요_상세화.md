# 📍 Phase 2: 설교 개요(Outline) 상세화 (2차 요청)

## 🎯 단계 목표
Phase 1에서 선정된 본문과 주제를 바탕으로 설교의 뼈대가 되는 상세한 개요를 작성합니다.

## 📝 목사님 요청 예시
"좋아요. 이 본문과 주제로 개요를 작성해주세요. [특별 아이디어나 강조점 추가]"

---

## 📋 설교 개요 구조

### 1. 서론 (Introduction)

#### ① 시선 끌기 (Attention Getter)
**목적**: 청중의 주의를 사로잡고 설교에 집중하게 만들기

**방법**:
- 흥미로운 질문으로 시작
- 공감 가는 현대적 상황 제시
- 놀라운 통계나 사실
- 짧은 일화나 스토리

**예시**:
- "여러분은 지난 일주일 동안 몇 번이나 '아, 이건 아닌데...'라고 생각하셨나요?"
- "최근 연구에 따르면 현대인의 80%가 자신의 정체성에 대해 혼란을 느낀다고 합니다."

#### ② 배경 소개 (Context)
**목적**: "왜 지금 이 메시지인가?" 설명

**포함 내용**:
- 본문의 역사적/문화적 배경
- 현대 상황과의 연결점
- 이 시점에 이 메시지가 필요한 이유

**예시**:
- "우리가 오늘 보게 될 본문은 이스라엘이 바벨론 포로로 끌려간 시기에 기록되었습니다. 정체성을 잃고 방황하던 그 시대의 상황이 지금 우리의 모습과 놀랍도록 닮아있습니다."

#### ③ 주제 제시 (Thesis)
**목적**: 설교의 핵심 메시지를 한 문장으로 명확히 제시

**형식**: 
- 간결하고 기억하기 쉬운 문장
- 행동 지향적 표현
- 희망적이면서도 도전적인 톤

**예시**:
- "진짜 정체성은 세상이 아닌 하나님께서 부르시는 이름에 있습니다."
- "거룩함은 완벽함이 아니라, 하나님을 향해 나아가는 여정입니다."

---

### 2. 본론 (Body) - 3대지 구조

#### 대지 1: 진단 (원어/신학 심화 분석)
**목적**: 핵심 단어의 신학적/문화적 재정의를 통한 깊이 있는 이해

**구성 요소**:
1. **핵심 원어 제시**
   - 히브리어/헬라어 단어 소개
   - 발음 표기 (선택사항)
   
2. **원어의 의미 분석**
   - 사전적 정의
   - 문화적 배경
   - 당시 사람들의 이해
   
3. **신학적 함의**
   - 이 단어가 담고 있는 신학적 깊이
   - 다른 성경 본문에서의 사용 예시
   
4. **현대적 재해석**
   - 우리 시대의 언어로 재정의
   - 오해되고 있는 부분 바로잡기

**예시 구조**:
```
대지 1: "거룩하라"는 무엇을 의미하는가?

1. 히브리어 '카도쉬(קָדוֹשׁ)'의 원래 의미
   - "구별되다", "분리되다"
   - 일반적인 것과 다른 특별한 것
   
2. 문화적 맥락
   - 성전 기물이 '카도쉬'
   - 제사장이 '카도쉬'
   - 안식일이 '카도쉬'
   
3. 신학적 의미
   - 하나님의 본질적 속성
   - 하나님께 속한 것의 표지
   
4. 현대적 적용
   - 거룩함 = 세상과 분리가 아닌, 하나님께 연결됨
   - 완벽함이 아닌, 하나님의 목적을 위해 구별됨
```

#### 대지 2: 심화 (기독론적/사중복음적 확장)
**목적**: 구약과 신약을 통합하여 예수 그리스도와 사중복음으로 연결

**구성 요소**:
1. **신구약 연결**
   - 구약 본문과 예수 그리스도의 연결점
   - 예언과 성취의 관계
   
2. **사중복음 적용**
   - 중생: 이 본문이 말하는 새로운 탄생
   - 성결: 거룩함을 향한 성장의 과정
   - 신유: 영혼과 육체의 치유와 회복
   - 재림: 종말론적 소망과 완성
   
3. **십자가 복음**
   - 예수님의 십자가 사역과의 연결
   - 구원의 은혜와 감사

**예시 구조**:
```
대지 2: 예수 그리스도 안에서 완성되는 거룩함

1. 구약의 거룩 명령과 신약의 성취
   - 레위기 19:2 "너희는 거룩하라"
   - 히브리서 10:10 "우리가 거룩하게 되었느니라"
   
2. 사중복음의 렌즈
   - 중생: 거룩하게 될 수 있는 새 본성 (고후 5:17)
   - 성결: 거룩함을 향한 평생의 여정 (빌 3:12-14)
   - 신유: 죄로 병든 영혼의 치유 (벧전 2:24)
   - 재림: 완전한 거룩함의 완성 (요일 3:2)
   
3. 십자가의 은혜
   - 우리 힘으로는 불가능한 거룩함
   - 예수님의 보혈로 가능해진 거룩함
```

#### 대지 3: 적용 (청년/성도 맞춤형 실천 대안)
**목적**: 4인 페르소나가 각자의 삶에서 실천할 수 있는 구체적 대안 제시

**구성 요소**:
1. **작은 행동 (Small Action)**
   - 오늘 당장 시작할 수 있는 것
   - 측정 가능하고 구체적인 행동
   
2. **생활 밀착형 적용**
   - 직장에서
   - 가정에서
   - 관계에서
   - 개인의 시간에
   
3. **페르소나별 맞춤 적용**
   - 지혜: 틈새 시간 활용
   - 준서: 세상 vs 신앙의 통합
   - 민수: 구체적 변화의 단계
   - 수진: 일상 언어로 풀어낸 실천

**예시 구조**:
```
대지 3: 거룩함, 어떻게 시작할까?

1. 작은 행동: "하루 3분 구별의 시간"
   - 아침/점심/저녁 중 한 시간 선택
   - 3분간 하나님만 생각하는 시간
   
2. 구체적 적용
   
   **직장에서 (지혜를 위한)**
   - 회의 시작 전 30초 침묵 기도
   - "이 업무로 누구를 섬길 수 있을까?" 질문하기
   
   **관계에서 (준서를 위한)**
   - 하나님이 보시는 눈으로 사람 보기
   - 성공보다 성품을 우선순위에 두기
   
   **습관에서 (민수를 위한)**
   - 한 가지 나쁜 습관 대신 하나님 닮은 습관 선택
   - "왜 이게 어려운가?" 솔직한 대화
   
   **일상에서 (수진을 위한)**
   - 거룩함 = "하나님 전용 시간" 만들기
   - 시각화: 나의 하루 중 하나님 색깔 칠하기
```

### 3. 결론 (Conclusion)

#### ① 요약 (Summary)
**목적**: 설교의 핵심 포인트를 간결하게 정리

**방법**:
- 3대지의 핵심을 각 1문장으로 요약
- 주제문(Thesis)을 다시 강조
- 청중이 기억해야 할 핵심 키워드 반복

**예시**:
```
오늘 우리는 세 가지를 배웠습니다.
1) 거룩함은 완벽함이 아니라 구별됨입니다.
2) 예수님의 십자가로 우리는 거룩하게 될 수 있습니다.
3) 작은 행동부터 시작하면 변화가 일어납니다.
```

#### ② 결단과 도전 (Call to Action)
**목적**: 청중이 구체적으로 결단하고 행동하도록 촉구

**포함 요소**:
- 명확한 행동 제안
- 긴급성 (왜 지금인가?)
- 희망의 메시지 (할 수 있다!)
- 성령의 도우심 강조

**예시**:
```
이번 주, 여러분은 어떤 한 가지 결단을 하시겠습니까?
- 하루 3분 하나님만을 위한 시간을 낼 것인가요?
- 회의 시작 전 30초 기도를 시작할 것인가요?
- 한 가지 나쁜 습관을 하나님 닮은 습관으로 바꿀 것인가요?

작은 것부터 시작하세요. 성령님께서 함께 하실 것입니다.
```

#### ③ 기도문 (Prayer)
**목적**: 설교의 내용을 기도로 승화시키고, 청중과 함께 하나님께 나아감

**구성**:
- 서론에서 제기한 문제를 기도로 다시 가져옴
- 3대지의 내용을 고백과 간구로 표현
- 구체적인 적용 항목을 기도 제목으로 제시
- 성령의 도우심과 은혜를 간구

**형식**:
```
"사랑하는 하나님 아버지,

[서론의 문제 의식]
오늘 우리는 [문제/고민]으로 주님 앞에 나왔습니다.

[대지 1-2의 내용을 고백으로]
주님께서 [진리/복음]을 깨닫게 하셨습니다.
예수님의 [십자가/은혜]로 우리가 [변화/구원]받았음을 고백합니다.

[대지 3의 적용을 간구로]
이제 우리가 [구체적 행동]을 시작할 수 있도록 도와주세요.
[페르소나별 기도]
- 바쁜 우리가 틈새 시간을 드릴 수 있도록
- 혼란한 우리에게 정체성을 확고히 하도록
- 변화 없는 우리의 고집을 꺾어주시도록
- 낯선 용어를 이해하고 실천하도록

[마무리 - 성령의 도우심]
성령님, 우리와 함께 하셔서 이 결단을 지켜주세요.

예수님의 이름으로 기도드립니다. 아멘."
```

## 💡 윤비서 작업 팁

1. **균형 잡힌 길이**: 각 대지가 비슷한 무게감을 가지도록 조정
2. **원어의 깊이**: 대지 1에서 최소 1-2개의 핵심 원어를 깊이 있게 다룰 것
3. **사중복음 필수**: 대지 2에서 사중복음 4가지 요소를 반드시 언급
4. **구체성**: 대지 3에서는 추상적 권면보다 구체적 행동 제시
5. **기도문 정성**: 결론의 기도문은 설교 전체를 압축한 완성된 형태로 작성
