## 📝 목사님 요청 유형

### 유형 1: 부분 수정 요청
```
예시:
"대지 2의 신유 부분을 좀 더 구체적으로 풀어줘"
"서론이 너무 길어. 3분 정도로 줄여줘"
"민수를 위한 적용이 아직 추상적이야. 더 구체적으로"
```

### 유형 2: 전체 톤 조정
```
예시:
"전체적으로 좀 더 도전적인 톤으로"
"청년들에게 더 친근하게"
"복음의 날카로움을 더 강화해줘"
```

### 유형 3: 내용 추가/삭제
```
예시:
"재림에 대한 부분을 더 추가해줘"
"이 예화는 빼고 다른 걸로"
"여기에 십자가 복음을 더 강조해줘"
```

## 🔧 수정 작업 프로세스

### 1단계: 피드백 분석

#### 피드백 분류
```

1. [부분]: [구체적 수정 내용]

3. ...

🎨 톤/스타일 조정:
- [조정 방향]

➕ 추가 요청:
- [추가할 내용]

➖ 삭제 요청:
- [삭제할 내용]

⏱️ 시간 조정:
- [늘려야 할 부분 / 줄여야 할 부분]
```

### 2단계: 우선순위 결정

**우선순위 원칙**:
1. 신학적 정확성 관련 → 최우선
2. 메시지 명료성 관련 → 우선
3. 적용의 구체성 → 중요
4. 표현/스타일 → 마지막

### 3단계: 수정 실행

#### 부분 수정 시
```
[수정 전]
해당 부분의 원문

[수정 후]
개선된 내용

[수정 사유]
왜 이렇게 수정했는지 간략히 설명
```

#### 전체 톤 조정 시
- 전체 원고를 다시 검토하며 일관된 톤으로 조정
- 특히 서론, 각 대지의 시작 부분, 결론에 집중
- 수정된 부분을 하이라이트하여 표시

### 4단계: 재검토

**수정 후 필수 체크**:
- [ ] 목사님의 모든 피드백이 반영되었는가?
- [ ] 수정으로 인해 다른 부분과 불일치가 생기지 않았는가?
- [ ] 전체 흐름이 여전히 자연스러운가?
- [ ] 시간 배분이 적절한가?

## 📝 한 줄 요약 작성

### 한 줄 요약의 목적
- 설교의 핵심을 압축적으로 표현
- SNS, 주보, 광고 등에 활용
- 청중이 쉽게 기억하고 공유할 수 있도록

### 한 줄 요약 작성 원칙
1. **간결함**: 15-20자 이내
2. **명료함**: 주제가 명확히 드러나야 함
3. **기억하기 쉬움**: 리듬감 있거나 인상적인 표현
4. **행동 지향적**: 가능하면 실천을 암시
5. **복음 중심**: 은혜가 느껴지는 표현

### 한 줄 요약 형식 예시

#### 형식 1: 대조 구조
```
"완벽함이 아닌 구별됨, 이것이 거룩입니다"
"내 노력이 아닌 하나님 은혜, 이것이 변화입니다"
```

#### 형식 2: 질문-답변
```
"어떻게 변할까? 작은 순종으로 시작하세요"
"왜 거룩해야 하나? 하나님께 속했기 때문입니다"
```

#### 형식 3: 명령-약속
```
"3분을 드리세요, 하나님이 일하십니다"
"작게 시작하세요, 크게 바뀔 것입니다"
```

#### 형식 4: 핵심 키워드 강조
```
"새 마음, 새 시작, 새 삶"
"레브 하다쉬 - 하나님이 주시는 새 마음"
```

## 📚 부록: 자주 하는 수정 요청 대응 가이드

### "너무 길어요"
- 예화를 줄이거나 하나 삭제
- 각 대지의 부연 설명 축약
- 결론 부분 간결화

### "너무 짧아요"
- 원어 설명 더 상세화
- 적용 예시 추가 (페르소나별)
- 성경 구절 인용 확대

### "너무 어려워요"
- 신학 용어 추가 설명
- 브릿지 언어 더 활용
- 일상적 예시 추가

### "너무 쉬워요"
- 원어 분석 깊이 추가
- 신학적 함의 확장
- 학술적 통찰 포함 (적절히)

### "도전이 약해요"
- 죄의 직면 강화
- 회개의 필요성 명확화
- 온전한 헌신 도전 추가

### "은혜가 부족해요"
- 십자가 복음 강조
- 하나님의 먼저 하신 사랑 부각
- "할 수 있다" 메시지 강화
