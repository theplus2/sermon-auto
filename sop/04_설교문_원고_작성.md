# 📍 Phase 4: 설교문 원고 작성 (3차 요청)

## 🎯 단계 목표
Phase 2의 개요와 Phase 3의 피드백을 반영하여 완성도 높은 설교 원고를 작성합니다.

---

## 📝 목사님 요청 예시
"피드백 반영해서 풍성한 원고로 작성해줘."

---

## 📋 원고 작성 프로세스

### 1단계: 피드백 검토 및 반영 계획

#### 반영해야 할 주요 피드백 정리
1. **신학적 보완 사항**
   - 사중복음 요소 강화 필요 부분
   - 기독론적 연결 추가 필요 부분
   
2. **청중 피드백 반영**
   - 지혜: 구체적 적용 더 필요
   - 준서: 세상과 신앙의 연결 강화
   - 민수: 변화의 단계적 로드맵 제시
   - 수진: 브릿지 언어 적극 활용
   
3. **목회적 방어 포인트**
   - 복음의 날카로움 유지
   - 용어의 품격 지키기

### 2단계: 원고화 스타일 가이드

#### 📝 문장 작성 원칙

**① 짧고 간결한 문장**
```
❌ 나쁜 예: 
"우리가 살아가면서 하나님을 섬기는 것은 단순히 교회에 나와서 예배를 드리고 
봉사를 하는 것만이 아니라, 우리의 일상 속에서 매 순간 하나님을 생각하며 
그분의 뜻을 따라 행동하는 것을 의미하는 것입니다."

✅ 좋은 예:
"하나님을 섬기는 것은 예배만이 아닙니다. 
일상의 매 순간, 그분의 뜻을 따라 사는 것입니다."
```

**② 능동적이고 현재형 표현**
```
❌ 나쁜 예: "~해야 할 것입니다", "~하는 것이 필요합니다"
✅ 좋은 예: "~합니다", "~하세요"
```

**③ 청중과의 대화체**
```
❌ 나쁜 예: "본 설교자는 이렇게 생각한다"
✅ 좋은 예: "여러분, 우리 함께 생각해봅시다"
```

#### 📖 성경 인용 형식

**표준 형식**: **[장 절 "인용 내용"]**

```
예시:
에스겔은 이스라엘 백성에게 선포합니다. 
[에스겔 36:26 "또 새 영을 너희 속에 두고 새 마음을 너희에게 주되 
너희 육신에서 굳은 마음을 제거하고 부드러운 마음을 줄 것이며"]

짧은 인용:
바울은 [고린도전서 6:19 "너희 몸은 성령의 전"]이라고 말합니다.
```

#### 💬 신학 용어 사용 원칙

**원칙**: 원어의 품격 유지 + 현대적 재정의

```
1차 언급 시: 용어 + 설명
"성화, 즉 하나님을 닮아가는 평생의 여정은..."

2차 이후: 용어만 사용 또는 일상 언어 혼용
"이 성화의 과정에서..." 또는 "하나님을 닮아가는 이 여정에서..."

필수 신학 용어:
- 중생, 성결, 성화, 칭의, 구원, 거룩함, 회개, 헌신
→ 반드시 원어와 함께 정의 후 사용

브릿지 언어 활용:
- Phase 3에서 제안된 일상 언어를 적극 활용
```

#### 🎨 비유와 예화 사용

**원칙**: 구체적이고 공감 가능한 현대적 예시

```
✅ 좋은 예:
"성화는 마치 휴대폰을 업데이트하는 것과 같습니다. 
갑자기 완전히 바뀌지 않습니다. 
하지만 업데이트를 거듭할수록 버그가 줄고, 
새로운 기능이 추가되고, 더 나은 버전이 됩니다."

❌ 피해야 할 예:
- 너무 오래된 예화 (80-90년대 이야기)
- 특정 계층만 이해할 수 있는 전문적 예시
- 지나치게 긴 이야기
```

### 3단계: 원고 구조별 작성 가이드

#### 🌟 서론 작성 (5-7분)

**목표**: 청중의 마음을 사로잡고 본론으로 자연스럽게 연결

**구성**:
1. **훅(Hook)**: 30초-1분
   - 질문, 통계, 짧은 이야기, 공감 가는 상황
   
2. **배경(Context)**: 2-3분
   - 본문의 역사적/문화적 배경
   - "왜 지금 이 메시지인가?"
   
3. **주제 제시(Thesis)**: 1분
   - 핵심 메시지 한 문장
   - 설교의 방향 제시

**작성 예시**:
```
[훅]
여러분은 지난 일주일 동안 스스로에게 몇 번이나 실망하셨나요? 
(잠깐 침묵)
"이번엔 다를 거야" 결심했던 것들, 며칠 못 가 무너졌나요?

[배경]
오늘 우리가 볼 본문은 바벨론 포로로 끌려간 이스라엘 백성에게 
주신 하나님의 메시지입니다. 
그들도 수없이 실패하고, 좌절하고, 자신들의 정체성조차 잃어버렸습니다.
그런 그들에게 하나님은 놀라운 약속을 하십니다.

[주제 제시]
오늘 우리는 이렇게 배울 것입니다:
"진짜 변화는 내 노력이 아니라, 하나님의 은혜로 시작됩니다."
```

#### 📚 본론 작성 (각 대지 7-10분)

**대지 1: 원어 분석 및 신학적 재정의**

**구성**:
1. 대지 제목 (기억하기 쉽게)
2. 핵심 원어 소개 (히브리어/헬라어)
3. 원어의 의미 설명 (사전적 + 문화적)
4. 신학적 함의
5. 현대적 재정의
6. 소결론 (한 문장으로 요약)

**작성 가이드**:
```
대지 제목: "새 마음, 새 시작"

[원어 소개]
하나님은 에스겔을 통해 "레브 하다쉬(לֵב חָדָשׁ)", 
즉 '새 마음'을 주시겠다고 약속하십니다.

[원어 의미]
히브리어 '레브'는 단순히 감정의 기관이 아닙니다. 
고대 히브리인들에게 '레브'는 생각, 의지, 감정이 모두 통합된 
인격의 중심이었습니다.

[신학적 함의]
따라서 '새 레브'는 단순히 "마음을 바꾸자"는 게 아닙니다.
우리 존재의 핵심, 생각과 의지와 감정 전체가 새로워지는 것입니다.

[현대적 재정의]
오늘날 우리 언어로 하면, 
'새 마음'은 '새로운 운영체제'를 설치하는 것과 같습니다.
단순한 앱 하나를 지우고 깔고가 아니라, 
시스템 전체가 업그레이드되는 것이죠.

[소결론]
하나님이 주시는 변화는 표면적 행동 수정이 아닙니다.
존재의 핵심부터 완전히 새롭게 하시는 것입니다.
```

**대지 2: 기독론적/사중복음적 확장**

**구성**:
1. 대지 제목
2. 구약 본문과 예수 그리스도의 연결
3. 사중복음 적용 (중생, 성결, 신유, 재림)
4. 십자가 복음의 은혜
5. 소결론

**작성 가이드**:
```
대지 제목: "예수님 안에서 이루어지는 약속"

[기독론적 연결]
에스겔이 예언한 이 '새 마음'은 누구를 통해 실현됩니까?
예수 그리스도의 십자가와 부활을 통해서입니다.

[사중복음]
• 중생: [고린도후서 5:17 "누구든지 그리스도 안에 있으면 
새로운 피조물이라"] 
새 마음은 거듭남의 순간에 우리 안에 심어집니다.

• 성결: [빌립보서 2:13 "너희 안에서 행하시는 이는 하나님이시니"] 
그 새 마음이 자라나도록 성령께서 매일 일하십니다.

• 신유: [베드로전서 2:24 "그가 친히 나무에 달려 
그 몸으로 우리 죄를 담당하셨으니"]
죄로 굳어진 우리 마음을 치유하십니다.

• 재림: [요한일서 3:2 "그가 나타나시면 우리가 그와 같을 줄을 아는 것은"]
완전한 새 마음은 예수님 재림 때 완성됩니다.

[십자가 은혜]
이 모든 것이 우리 노력으로 되는 게 아닙니다.
예수님이 십자가에서 흘리신 보혈로 가능해진 은혜입니다.

[소결론]
새 마음의 약속은 예수 그리스도 안에서 
이미 시작되었고, 지금 진행 중이며, 
재림 때 완성될 것입니다.
```

**대지 3: 구체적 적용**

**구성**:
1. 대지 제목
2. "작은 행동" 제시
3. 페르소나별 맞춤 적용
4. 장애물과 극복 방법
5. 격려와 도전
6. 소결론

**작성 가이드**:
```
대지 제목: "오늘부터 시작하는 새 마음"

[작은 행동]
그렇다면 우리는 어떻게 시작할까요?
아주 작은 것부터 시작하십시오.
'하루 3분 하나님 전용 시간'을 만드는 것입니다.

[페르소나별 적용]
지혜 씨처럼 바쁜 워킹맘이시라면,
아침 출근 준비하며 커피 한 잔 마실 때, 그 3분을 드리세요.
"하나님, 오늘 회의에서 인내하게 하소서"

준서처럼 미래를 고민하는 청년이라면,
점심시간 식사 후, 커피 마시며 그 3분을 드리세요.
"하나님, 오늘 내 선택이 성공보다 성품을 따르게 하소서"

민수처럼 변화가 어려운 분이라면,
저녁 퇴근 후 집에 도착해서, 옷 갈아입기 전 그 3분을 드리세요.
"하나님, 오늘 내 고집 센 자아가 조금 꺾였습니다. 
내일도 도와주세요"

수진처럼 신앙이 새로운 분이라면,
하루 중 가장 여유로운 시간, 그 3분을 드리세요.
"하나님, 당신을 알아가는 이 시간이 좋습니다"

[장애물과 극복]
"바빠서 못 하겠어요"라고 생각하실 수 있습니다.
하지만 생각해보세요. 
우리는 SNS 보는 데 하루에 몇 분을 쓰나요?
3분은 가능합니다.

[격려와 도전]
이 작은 3분이 여러분의 마음을 바꿉니다.
하루 3분, 일주일이면 21분입니다.
한 달이면 90분, 1시간 반입니다.

하지만 중요한 건 시간이 아닙니다.
하나님께 제 시간을 드리는 그 결단입니다.

[소결론]
새 마음은 거창한 결심이 아니라, 
작은 순종에서 시작됩니다.
```

#### 🎬 결론 작성 (5-7분)

**구성**:
1. 요약 (3대지 핵심)
2. 결단과 도전
3. 구체적 행동 제안
4. 희망의 메시지
5. 기도문

**작성 가이드**:
```
[요약]
오늘 우리는 세 가지를 배웠습니다.

첫째, 하나님이 약속하신 새 마음은 
우리 존재의 핵심을 완전히 새롭게 하는 것입니다.

둘째, 이 약속은 예수 그리스도의 십자가를 통해 
이미 우리에게 주어졌고, 
성령께서 지금 이 순간에도 우리 안에서 일하고 계십니다.

셋째, 우리는 작은 순종으로 이 새 마음의 여정을 
오늘 당장 시작할 수 있습니다.

[결단과 도전]
이제 결단할 시간입니다.
여러분은 이번 주에 무엇을 하시겠습니까?

하루 3분 하나님 전용 시간을 만드시겠습니까?
아니면 계속 "바쁘다"는 핑계로 미루시겠습니까?

[구체적 행동]
오늘 집에 가시면, 
휴대폰에 알람을 하나 설정하세요.
제목은 "하나님 3분"입니다.
매일 같은 시간에 울리도록 설정하세요.

그리고 그 시간이 되면, 
무엇을 하고 있든 잠깐 멈추고,
하나님께 말을 거세요.

[희망의 메시지]
여러분, 걱정하지 마세요.
완벽하게 하려고 하지 마세요.
어떤 날은 잊어버릴 수도 있습니다.
어떤 날은 형식적일 수도 있습니다.

괜찮습니다.
하나님은 우리의 완벽함을 원하지 않으십니다.
우리의 진심을 원하십니다.

작은 순종, 작은 시간이 쌓여서
어느새 여러분은 변화되어 있을 것입니다.

성령님께서 여러분과 함께 하실 것입니다.

[기도문으로 마무리]
(다음 섹션 참조)
```

### 4단계: 기도문 작성 가이드

#### 기도문 구조
1. **하나님께 나아감** (호칭)
2. **설교 내용 고백** (진리 인정)
3. **회개와 감사** (마음의 반응)
4. **구체적 간구** (적용 기도)
5. **성령의 도우심 간구**
6. **마무리** (예수님의 이름으로)

#### 기도문 작성 예시
```
함께 기도하겠습니다.

사랑하는 하나님 아버지,

오늘 우리는 주님께서 약속하신 새 마음에 대해 배웠습니다.
우리 스스로는 절대 바뀔 수 없는 굳은 마음을 
부드러운 마음으로 바꾸어 주시겠다는 그 놀라운 약속을 들었습니다.

주님, 고백합니다.
우리는 수없이 실패했습니다.
"이번에는 다를 거야" 결심했지만, 
며칠 못 가 무너지고 말았습니다.

그런 우리에게 
예수 그리스도의 십자가를 통해 
이미 새 마음을 주셨음을 감사드립니다.

이제 우리가 작은 순종으로 반응하게 하소서.

바쁜 지혜가 
출근길 커피 한 잔 마실 때 
주님께 3분을 드릴 수 있도록,

방황하는 준서가
점심 식사 후 잠시 멈춰 서서 
주님께 인생의 방향을 물을 수 있도록,

변화가 어려운 민수가
퇴근 후 집에 도착해 
주님께 자신의 고집을 내려놓을 수 있도록,

신앙이 새로운 수진이
하루 중 가장 여유로운 시간에 
주님과 대화하는 기쁨을 누릴 수 있도록,

성령님, 우리와 함께 하소서.

우리의 작은 순종을 보시고 
크신 은혜로 역사하소서.

우리 스스로 할 수 없는 변화를 
주님께서 이루어 주소서.

예수 그리스도의 이름으로 기도드립니다.
아멘.
```

### 5단계: 5대 핵심 질문 체크리스트

원고 작성 완료 후, 반드시 다음 질문들을 점검하세요:

#### ❓ 질문 1: 메시지가 한 문장으로 명료한가?
```
✅ 자가 점검:
- 설교를 듣고 나서 청중이 한 문장으로 요약할 수 있는가?
- 주제문이 명확하게 제시되었는가?
- 3대지가 모두 이 주제문을 지지하는가?
```

#### ❓ 질문 2: 왜 지금 이 내용을 알아야 하는가? (당위성)
```
✅ 자가 점검:
- 서론에서 "왜 지금인가?"를 명확히 설명했는가?
- 현대 청중의 삶과 연결되는가?
- 긴급성이 느껴지는가?
```

#### ❓ 질문 3: '아주 작은 행동'이 제시되었는가? (적용)
```
✅ 자가 점검:
- 대지 3에서 구체적이고 실천 가능한 행동을 제시했는가?
- "오늘 당장" 시작할 수 있는 것인가?
- 측정 가능한가? (예: 3분, 한 가지, 매일 등)
```

#### ❓ 질문 4: 복음적 이유가 충분한가? (근거)
```
✅ 자가 점검:
- "해야 한다"가 아니라 "할 수 있다" (은혜)로 설명했는가?
- 예수 그리스도의 십자가와 연결되었는가?
- 사중복음이 명확히 드러나는가?
```

#### ❓ 질문 5: 기억에 남을 키워드/이미지가 있는가? (기억 장치)
```
✅ 자가 점검:
- 한 주 후에도 기억날 만한 표현이 있는가?
- 시각적 이미지나 비유가 효과적인가?
- 반복되는 키워드가 있는가?

예시:
- "하루 3분 하나님 전용 시간"
- "새 마음 = 새 운영체제"
- "레브 하다쉬"
```
